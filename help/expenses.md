# Expense Tracker

Log daily spending, categorize expenses, convert amounts to USD using offline exchange rates, and export a CSV report. Everything works offline — no bank connections, no cloud sync.

---

## How to add an expense

1. Click **Expenses** in the sidebar.
2. Click the **+ Add** tab.
3. Enter the amount and select the currency you paid in.
4. Select a category.
5. Add an optional description.
6. Set the date — defaults to today.
7. Click **Save Expense**.

The entry appears immediately in the All Expenses list.

---

## Categories

| Category | Examples |
|---|---|
| Accommodation | Hotels, hostels, Airbnb |
| Food & Drink | Restaurants, cafés, groceries |
| Transport | Flights, trains, taxis, rideshare |
| Activities | Tours, entry fees, experiences |
| Shopping | Clothing, gear, souvenirs |
| Health | Pharmacy, doctor, insurance |
| Communications | SIM cards, data plans, calls |
| Visa & Admin | Visa fees, travel insurance, notary |
| Other | Anything that does not fit above |

---

## Currency conversion

Expenses are logged in the currency you paid in. Nomad Sentinel converts amounts to USD for totals and summaries using offline exchange rate tables bundled with the app.

Exchange rates are updated with each Nomad Sentinel release. They are not live rates — treat converted totals as estimates, not exact figures.

---

## Summary view

1. Click the **Summary** tab.
2. Totals are shown by category with a proportional bar chart.
3. All amounts are converted to USD for comparison.

---

## Deleting an expense

1. Find the expense in the **All Expenses** list.
2. Click **✕** on the right side of the entry.
3. Deletion is immediate and permanent.

---

## Exporting to CSV

1. Click **Export CSV** in the top right of the Expenses page.
2. The file is written to your app data directory.
3. The full file path is shown in the confirmation dialog.
4. Open the file in any spreadsheet application.
