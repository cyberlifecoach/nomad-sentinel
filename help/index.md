# Nomad Sentinel Help

**Plan smarter. Travel safer. Stay private — anywhere.**

Nomad Sentinel is a privacy-first desktop app for digital nomads, journalists, and remote workers. All data is stored locally on your device, encrypted at rest. No cloud accounts. No telemetry. No internet required.

---

## Features

| Feature | What it does |
|---|---|
| [Encrypted Journal](journal.md) | Write private entries encrypted with your passphrase |
| [Secure Setup Wizard](setup-wizard.md) | Harden your devices before you travel |
| [Emergency Toolkit](emergency.md) | Encrypted contacts, exit checklists, secure deletion |
| [Metadata Scrubber](metadata-scrubber.md) | Strip GPS and EXIF data from photos before sharing |
| [Packing Checklists](packing-checklists.md) | Pre-built and custom packing lists by trip type |
| [Country Security Checklists](country-checklists.md) | Border rules, scam alerts, SIM laws for 18 countries |
| [Phrasebook](phrasebook.md) | Essential phrases in 20+ languages, offline |
| [Expense Tracker](expenses.md) | Log spending, convert currencies, export to CSV |
| [Settings & Profiles](profiles.md) | Multiple traveler profiles with separate passphrases |
| [Backup & Export](backup.md) | Encrypted archive of all your data |

---

## Key principles

**Offline first.** Every feature works without an internet connection. The only exception is the optional FX rate refresh in the Expense Tracker.

**Encrypted at rest.** Your journal entries, emergency contacts, and profile data are encrypted using AES-256-GCM. Your passphrase never leaves your device.

**No tracking.** Nomad Sentinel contains zero analytics, crash reporting, or usage telemetry of any kind.
